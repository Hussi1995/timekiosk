TimeKiosk Pro (Vite + React + TypeScript)

Slik kjører du prosjektet:
1. Pakk ut ZIP-filen
2. Åpne terminal i mappen
3. Kjør: npm install
4. Kjør: npm run dev
5. Åpne adressen Vite viser i terminalen

Viktig:
- Ikke åpne src/App.tsx eller index.tsx direkte i nettleseren.
- TSX må kjøres gjennom Vite.
