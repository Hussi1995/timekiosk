# TimeKiosk Pro

Dette er en kjørbar Vite + React + TypeScript-app for enkel timeregistrering på iPad eller PC.

## Start

```bash
npm install
npm run dev
```

## Viktig

Ikke åpne `.tsx`-filer direkte i nettleseren. Prosjektet må kjøres med Vite.
